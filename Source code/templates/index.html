Intro to QueryAI

The QueryAI project aims to streamline and enhance the process of accessing and understanding client data stored in internal databases through a user-friendly chat interface. Leveraging a Large Language Model (LLM), this AI-powered solution allows organization members to query the database using natural language and receive accurate, contextually relevant responses, complete with interactive data visualizations, proactive insights, and personalized recommendations.
Steps to work with QueryAI:
  • Step 1: Click on Get Started.
  • Step 2: Enter Database Connection Details such as Host name, User, Password and Database name.
  • Step 3: Verify the tables present in your database and click on Continue to ask question button.
  • Step 4: Enter your question releted your database and click on submit.
  • Step 5: The results for the given question will be shown in the table form. Then you can download the result or you can visualize it.
  • Step 6: For visualization click on Visualize Result select the visualization techniques select x-axis and y-axis attributes and click on Generate Charts. Then all the selected graphs are plotted and displayed you can download it for further use.
